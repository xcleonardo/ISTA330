In this practicum you will develop a REST API using Nodejs and Express. You will update your Imagequiz application that you developed in Practicum 6 to use this API to fetch your data from a server. You will deploy the REST API server on Heroku.
  • First develop a REST API for your imagequiz web application. The API should have the following methods:
    1. '/quizzes' - a GET method returning the meta data about all the quizzes
    2. '/quiz/:id' - a GET method returning the quiz number (:id) questions
    3. '/score' - a POST method storing the score for a quiz taken by a user
  • Then deploy your REST API to Heroku. You should follow the follwoing naming convention for your Heroku deployment: the name of your Heroku application should be "yourgithubusername-imagequiz.herokuapp.com".
  • Then update your Imagequiz web application to get your data from the REST API you deployed on Heroku.
For more information on how to do this practicum refer to the lectures on November 17th and November 19th.
